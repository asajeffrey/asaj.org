This document describes a categorically motivated graphical presentation of the semantics of programs, including an implementation as a Java applet.

  • Premonoidal categories and a graphical view of programs describes a correspondence between a categorical and a graphical semantics of programs. This is also available in PostScript in two parts: the main text and the appendices.

  • The slides from a talk on the subject, also available as PostScript.

  • A flow graph applet gives a demonstration of the features of the applet.

  • Source code and Javadoc documentation are also available.

  • You can also download this whole site, including the source code and documentation, as a 700k compressed tar file or as a 1100k zip file.

These papers contain Java 1.1, so should be viewed in a recent browser such as Netscape 4.5 or Internet Explorer 4. The applet was compiled using the freeware Kaffe compiler, which in turn uses the Pizza compiler.
