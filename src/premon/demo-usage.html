Usage

To embed a graph into an HTML document, you use an <applet> tag with attributes:

  • code="AppletPremon.class"
  • width=width in pixels
  • height=height in pixels
and parameters:
  • <param name="exp" value="term">
  • <param name="base" value="constructors"> (optional)
  • <param name="free" value="context"> (optional)
For example the graph:
is generated by: 
   <applet code="AppletPremon.class" width=100 height=120>
   <param name="base" value="
      val * (int, int) : int;
      val + (int, int) : int;
      val - (int, int) : int;
      val / (int, int) : int;
   ">
   <param name="exp" value="
      (1+2)*3
   ">
   </applet>
